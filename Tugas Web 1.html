TEKNIK INFORMATIKA UNAI

APA ITU TEKNIK INFORMATIKA

PENGERTIAN TEKNIK INFORMATIKA

Teknik Informatika merupakan salah satu jurusan perkuliahan
dengan peminat terbanyak dikarenakan peran teknologi yang
begitu berpengaruh di dalam dunia kerja.

APA SAJA YANG DIPELAJARI DI TEKNIK INFORMATIKA

1. Bahasa Pemrograman

Tulisan-tulisan yang kita lihat dimonitor device kita dapat
kita pahami dengan mudah, namun dibalik itu ada sebuah bahasa
pemrograman yang bekerja dibalik itu semua. Bahasa pemrograman
memungkinkan kita menginput bahasa yang dapat dimengerti
komputer. Ada banyak sekali bahasa pemrograman, diantaranya
adalah:

a. C++
C++ adalah bahasa pemrograman paling umum yang merupakan
pengembangan dari bahasa C. Di awal perkuliahan, bahasa ini
adalah bahasa yang harusnya dapat dikenal mahasiswa
b. Java
Java adalah bahasa pemrograman yang berfokus pada pemrograman
berorientasi objek. Sebagai sebuah platform, Java terbagi menjadi
dua bagian, yaitu:
i) Java Virtual Main Machine (JVM)
ii) Java Application Programming Interface (JavaAPI)
c. Phyton
Bahasa pemrograman phyton adalah bahasa pemrograman yang
paling mudah dipahami, oleh karenanya bahasa ini dinilai lebih
mudah untuk mengembangkan situs web dan membuat algoritma.

2. Algoritma

Algoritma merupakan prosedur, langkah-langkah, ataupun
pengumpulan suatu data yang tersusun dan terstruktur untuk
menjalankan sebuah atau beberapa program dari rangkaian instruksi
yang telah diberikan.

Dalam sebuah pemrograman, algoritma berfungsi untuk mengatur
bagaimana sebuah program nantinya akan menjalankan fungsinya

3. Ilmu dasar komputer

Walaupun seorang programmer hanya berfokus pada perangkat lunak,
tetapi programmer setidaknya harus mengetahui dasar-dasar perangkat
lunak komputer, hal ini dikarenakan perangkat lunak dan perangkat
merupakan komponen yang tidak dapat saling dipisahkan.

dan lain sebagainya

BAGAIMANA MASA DEPAN LULUSAN TEKNIK INFORMATIKA

1. Programmer

Programmer merupakan profesi yang bertujuan untuk untuk membuat,
merancang, dan memperbaiki program menggunakan bahasa pemrograman.
Melalui program yang dibuat nantinya akan berguna bagi sebuah
institusi atau perusahaan.

2. System Analisyst

Profesi ini bertugas untuk menganalisa sistem yang nantinya akan
digunakan pada sebuah perusahaan

3. Dosen IT

Seorang lulusan teknik Informatika tak hanya berurusan degan dunia
perkantoran atau perusahaan. Tetapi juga dapat bekerja sebagai
dosen atau pengajar IT ada sebuah sekolah atau sekolah tinggi.
